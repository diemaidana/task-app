# Task App

### Introduction
The Task Tracking application is an educational one. The main objetive of this repository is to keep building backend
applications and keep learning Spring Boot.

### CRUD endpoints

* `POST` mapping to create a task.
* `GET` mapping to list all tasks.
* `PUT` mapping to update a task.
* `DELETE` mapping to delete a task.

### Instalation

1.  **Clone repository**
    ```sh
    $ git clone https://github.com/diemaidana/task-app
    ```

2. **Run Docker**
    ```sh
    $ docker-compose up
    ```

3. **Run Task App**
    ```sh
    $ mvn spring-boot:run
    ```

4. **Enter application in web browser**
    ```sh
   $ http://localhost:3000
    ```
